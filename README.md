# Employee Salary Data Analysis

## Objective
To analyze employee salary data using Python and extract insights using data visualization.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Loaded CSV data using Pandas
- Calculated average monthly salary
- Department-wise salary analysis
- Experience vs Salary analysis
- Correlation heatmap

## Dataset
The dataset contains employee details such as department, experience, age, city, gender, and monthly salary.

## Conclusion
The analysis shows that experience has a positive relationship with salary and salary varies across departments.
